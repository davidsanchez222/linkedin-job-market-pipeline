This repo is from when I participated in UGAHacks 9 which is a hackathon my school hosts every year.
I created it two years ago and am now uploading it to GitHub.
The dataset includes structured job skill codes and unstructured descriptions; this pipeline extracts normalized skill tags from text to support analysis.

I added some example analytical SQL queries which are located at sql/queries.sql

## Environment Setup (Conda)

```bash
conda env create -f environment.yml
conda activate linkedin-etl
```
